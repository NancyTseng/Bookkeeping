# 記帳軟體
紀錄收入(薪水、股票)和支出(食衣住行育樂)，並計算單月剩餘存款

## 範例

SELECT * <br>
FROM `living expenses` <br>
WHERE date='2021-05-01' <br>
以上query可以查詢2021-05-01這一天的所有花費紀錄
